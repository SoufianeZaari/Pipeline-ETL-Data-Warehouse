# Pages Power BI - amélioration future

Le dashboard final officiel du projet est Metabase. Cette proposition Power BI est conservée comme piste future sous Windows.

## Page 1 - Vue générale

KPIs :

- chiffre d'affaires total ;
- nombre de commandes ;
- nombre de clients ;
- panier moyen ;
- taux de retour ;
- délai moyen de livraison.

Visuels :

- courbe du chiffre d'affaires par mois ;
- bar chart du chiffre d'affaires par région ;
- bar chart du chiffre d'affaires par catégorie ;
- carte ou tableau des top villes.

## Page 2 - Analyse clients

Visuels :

- top clients par chiffre d'affaires ;
- meilleurs clients à Tanger ;
- chiffre d'affaires par ville ;
- nombre de clients par région ;
- panier moyen par région.

Filtres recommandés :

- région ;
- ville ;
- trimestre ;
- catégorie produit.

## Page 3 - Analyse produits

Visuels :

- chiffre d'affaires par catégorie ;
- top produits vendus ;
- produits les plus retournés ;
- performance pendant Ramadan avec `dim_date[is_ramadan] = 1`.

Filtres recommandés :

- catégorie ;
- sous-catégorie ;
- marque ;
- période.

## Page 4 - Livraison & retours

Visuels :

- délai moyen de livraison ;
- taux de retour par région ;
- raisons de retour ;
- statut des livraisons ;
- performance par transporteur ;
- taux d'échec de livraison par transporteur.

Filtres recommandés :

- transporteur ;
- statut de livraison ;
- région ;
- période.
